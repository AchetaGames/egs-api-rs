# egs-api

A minimal asynchronous interface to Epic Games Store

## Current functionality
 - Authentication
 - Listing Assets
 - Get Asset metadata
 - Get Asset info
 - Get Ownership Token
 - Get Game Token
 - Get Entitlements
 - Get Library Items
 - Generate download links for chunks
